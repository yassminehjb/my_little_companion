# my_little_companion
THE BACKSTORY
My first submission got knocked back because it looked a bit too much like an
automated AI template. Honestly, that is completely fair. I used a ton of
hyper-clean loops, advanced classes, and complex grid math that looked way
too sterile.
To fix this, I threw out that entire codebase and rebuilt the backend by hand
using basic tools. It is a bit messy, but it works perfectly and is 100%
authentic.

HOW THE CODE IS HANDCRAFTED
No Framework Overkill: I removed all the advanced object-oriented
boilerplate. Everything runs on standard linear functions and simple global
state variables that are easy to track.
Manual Layout Layout: Instead of mathematically generating the interfaces
inside a wrapper, I wrote out the grid coordinate system position by
position. It took longer, but it ensures total control over the setup.
Multi-Threading: CustomTkinter locks up when using standard pauses. I used
basic native threads so the Pomodoro counter and progress bars update
without causing the script to hang.
APP FEATURES
Study Timer: A classic countdown clock that rewards your attention span
with direct experience points for your digital desk plant.
Task Manager: A simple input layout where you can log goals and append them
directly to an active workflow list.
Local Diary System: Writes your thoughts to a text file on your drive. A
basic dropdown module parses the file so you can pull up past entries
anytime.
Math Box: A traditional keypad configuration utilizing raw processing
evaluation to calculate string equations instantly.
Ambient Player: Displays standard playback progression tracking linked to a
native interactive sliding scale.
